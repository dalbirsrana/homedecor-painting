Data visualization means visually displaying data or using raw data into insights that can be easily interpreted by the viewer. <\/p>\n\n\n\n

\u201cWikipedia\u2019s definition of Data Visualization: Data visualization refers to the techniques used to communicate data or information by encoding it as visual objects (points, lines or bars) contained in graphics.\u201d<\/p><\/blockquote>\n\n\n\n

To be highly effective visualization, it is important to design the right visualization for the data so that users can interpret and make decisions based on what they observe.  <\/p>\n\n\n\n

General types of data visualization:<\/strong><\/p>\n\n\n\n

Charts, Tables, Graphs, Maps, Info-graphics, Dashboards<\/p>\n\n\n\n

More specific types of data visualization: <\/strong><\/p>\n\n\n\n

Area Chart, Bar Chart, Gantt Chart, Heat Map, Highlight Table, Bullet Graph, Circle View, Histogram, Matrix, Bubble Cloud, Dot Distribution Map,<\/p>\n\n\n\n

Data Visualization tools:<\/h3>\n\n\n\n

The most common tool for ordinary users is Excel. For more sophisticated abilities, we have Tableau, D3.js, Microsoft Power BI, Oracle Visual Analyzer, Google Charts etc.<\/p>\n\n\n\n

About D3.js<\/h3>\n\n\n\n

D3 produces interactive data visualization in web browsers. D3.js, as the name suggests (Data Driven Documents), it\u2019s a JavaScript library used to manipulate DOM objects using data. D3.js offers high flexibility and providing you to implement complex data. D3.js can be used for data range from few bits to gigabits.<\/p>\n\n\n\n

D3 uses HTML, SVG and CSS to bring data to life. D3 follows web standards and uses full capabilities of latest browsers. You can use D3 to generate an HTML table or create an interactive, dynamic SVG charts. The D3.js uses pre-build functions to select elements and create SVG objects. You can style or add transitions to these objects. D3 is extremely fast and supports large datasets in various formats such as JSON, CSV.<\/p>\n\n\n\n

Installing D3.js<\/h3>\n\n\n\n

You can install using npm (npm install d3) . You can also download or load directly from d3j.org. <\/p>\n\n\n\n
